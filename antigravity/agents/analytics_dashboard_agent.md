# Analytics Dashboard Agent

## Role

Responsible for visualization and analytics dashboards.

---

## Objective

- Display helmet compliance, traffic flow, speed, and zebra crossing violations
- Support queries from multiple cameras and sources
- Provide historical and real-time analytics

---

## Input

- Processed datasets from Data Engineering Agent
- Backend API data

---

## Technologies

- Grafana
- Apache Superset
- React (optional)
- PostgreSQL / InfluxDB

---

## Responsibilities

- Dashboard layout and design
- Query creation for analytics
- Integration with backend API
- Visualization of violations, speed distribution, and trajectories

---

## Output Deliverables

- Working dashboards
- Documentation for adding new metrics
- API integration instructions
