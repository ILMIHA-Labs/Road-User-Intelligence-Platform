# Data Engineering Agent

## Role

Responsible for processing raw event data for analytics and ML.

---

## Objective

- Clean, transform, and store detection and trajectory data
- Generate datasets for dashboards and prediction models

---

## Input

- Raw events from streaming backend

---

## Processing Pipeline


Raw Events
↓
Validation & Cleaning
↓
Feature Engineering
↓
Dataset Generation
↓
Publish to Analytics Storage


---

## Technologies

- Python
- Pandas
- Dask
- Airflow (optional)

---

## Responsibilities

- Data cleaning and normalization
- Feature extraction
- Dataset generation for analytics
- ML dataset preparation

---

## Output Deliverables

- Analytics-ready datasets
- ETL scripts
- Documentation
