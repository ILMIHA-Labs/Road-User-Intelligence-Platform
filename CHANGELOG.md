# Changelog

## 0.1.1 - 2026-05-23

### Added

- DPG-oriented governance, privacy, security, and open research release
  documentation
- maintainer-facing public release and DPG submission checklists
- lightweight CI for tests, governance-file presence, and repository hygiene

### Changed

- public runtime defaults now emphasize privacy-minimizing behavior
- `run_pipeline.sh` now requires an explicit `DEMO_VIDEO_SOURCE` for public use
- public docs now describe one canonical researcher workflow and one official
  backend-served dashboard surface
- canonical demo camera config no longer implies a bundled demo video is part
  of the guaranteed public package

### Removed

- tracked local runtime artifacts from the published repository surface
