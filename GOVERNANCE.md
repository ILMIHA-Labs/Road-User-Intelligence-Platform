# Governance

## Project role

Road User Intelligence Platform is maintained as an open-source
research-reference MVP for road-user analytics and crossing-safety monitoring.

## Stewardship

Project stewardship currently sits with:

- ILMIHA Labs
- Maintainer of record: Clement Ampofo

## Governance principles

The repository is governed by the following principles:

- road-safety relevance first
- privacy-minimizing defaults
- reproducibility for researchers
- transparent limitations and documented risk
- preference for open standards and platform independence

## Decision-making

The maintainer group is responsible for decisions on:

- roadmap direction
- public release readiness
- privacy and security defaults
- governance and policy documents
- acceptance or rejection of contributions
- release and submission readiness checklists

Changes with material policy impact should be reviewed against:

- `PRIVACY_POLICY.md`
- `SECURITY.md`
- `docs/data_governance.md`
- `docs/safety_and_risk.md`
- `docs/dpg_readiness.md`

## Release posture

This repository is published as:

- open-source software under the MIT license
- a research-reference MVP
- not a guarantee of production readiness for any specific deployment

## Community conduct

Community participation is governed by `CODE_OF_CONDUCT.md`.

## Escalation

Questions about governance, public release scope, or compliance positioning
should be directed to ILMIHA Labs and the repository maintainer of record.
